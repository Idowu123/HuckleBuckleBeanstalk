S/N:            Andela Project 3
Title:          HuckleBuckleBeanstalk 
Description:    This is a game in which the user guesses the number selcted by the computer.
                Each time the seeker makes a guess, the hider responds "getting hotter" if she's getting closer, or 
                "getting colder" if she's moved farther away. 
Live Preview:   http://idowu123.github.io/HuckleBuckleBeanstalk/

Outcome(s):     What I learnt: 
                1. Using javascript to restrict entries in a text field, 
                2. Using google fonts
                3. Binding the text field to the submit button using javascript
		4. Clearing a text field after an input is entered.
